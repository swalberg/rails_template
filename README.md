# Rails template

Just playing with the Rails Template system.

## Usage

`rails new testing -d sqlite3 -m rails_template/template.rb `

## Includes

* rspec
* twitter-bootstrap-rails
* sets up rvm
* Creates a git repo
* ActiveAdmin
